### Tend the garden

To create a deliberate bridge between two pieces, add a `links` list to a note's front matter using the destination's site path:

```yaml
links:
  - /posts/Who-Owns-The-Future/
  - /Content/Notes/AI-Superpowers/
```

Tags automatically appear as shared-topic connections. Use `links` for the more interesting question: *which idea does this note consciously continue, challenge, or illuminate?*
